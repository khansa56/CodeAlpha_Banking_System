# 🏦 Code Alpha Internship - Task 4
## Banking System (C++)

---

### 📌 Project Overview

A complete **Banking System** built in C++ using Object-Oriented Programming. This system allows creating customers, managing accounts, performing transactions, and tracking transaction history.

---

### ✨ Features

#### 👤 Customer Management
- ✅ Create new customers
- ✅ Store customer details (ID, Name, Phone, Email)
- ✅ View customer information

#### 💳 Account Management
- ✅ Create Savings/Current accounts
- ✅ Multiple accounts per customer
- ✅ View account information

#### 💰 Transactions
- ✅ Deposit money
- ✅ Withdraw money
- ✅ Transfer between accounts
- ✅ Balance validation

#### 📜 Transaction History
- ✅ Record all transactions
- ✅ View transaction history with date/time
- ✅ Track deposits, withdrawals, transfers

---

### 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| C++ | Core programming language |
| OOP | Classes (Customer, Account, Transaction) |
| Vectors | Dynamic data storage |
| iomanip | Formatted output |
| ctime | Date and time tracking |
